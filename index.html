Коротка інформація про GNU/Linux

GNU/Linux — це вільна і відкрита операційна система, яка поєднує ядро Linux з інструментами GNU. Вона широко використовується для серверів, настільних комп'ютерів, мобільних пристроїв та інших систем. Відмінна риса Linux — його надійність, гнучкість і підтримка спільнотою. Користувачі можуть вибирати з безлічі дистрибутивів, кожен з яких орієнтований на різні потреби.

Найбільш популярні дистрибутиви GNU/Linux

  • Ubuntu: Дружній до новачків дистрибутив, ідеальний для персональних комп'ютерів.
  • Debian: Стабільний і надійний, використовується на серверах і робочих станціях.
  • Fedora: Орієнтований на нові технології та розробників.
  • Arch Linux: Для просунутих користувачів, зосереджений на гнучкості та налаштуванні.
  • CentOS: Для серверів, відома своєю стабільністю.

Приклади команд роботи в Linux CLI

  1. pwd — Відображає поточну директорію.
  2. echo — Виводить текст або значення змінної.
  3. ls — Виводить список файлів у поточній директорії.
  4. cd — Змінює поточну директорію.
  5. touch — Створює новий порожній файл.
  6. mkdir — Створює нову директорію.
  7. cp — Копіює файли або директорії.
  8. mv — Переміщує або перейменовує файли/папки.
  9. rm — Видаляє файли або директорії.
  10. history — Відображає історію виконаних команд.

BASH-скрипт для імітації авторизації


#!/bin/bash

echo "Введіть ім'я користувача:"
read username

echo "Введіть пароль:"
read -s password

if [[ $username == "admin" && $password == "password123" ]]; then
    echo "Авторизація успішна!"
else
    echo "Невірне ім'я користувача або пароль!"
fi
